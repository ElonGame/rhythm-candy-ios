MusicPlayerDemo
===============
An iPhone OS 3.0 application to show some of the features of the Media Player Framework, esp. 
accessing the iPod media library and music playback.

Written by Ole Begemann, July 2009, http://oleb.net
Accompanying blog post: http://oleb.net/blog/2009/07/the-music-player-framework-in-the-iphone-sdk
